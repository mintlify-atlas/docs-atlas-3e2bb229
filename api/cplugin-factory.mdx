## CPluginFactory

`CPluginFactory` is the standard SDK implementation of `IPluginFactory3`. It is created by the `BEGIN_FACTORY` macro (and variants such as `BEGIN_FACTORY_CLASS`) inside `GetPluginFactory()`, but can also be subclassed when custom creation logic is needed. For a zero-overhead compile-time alternative that does not use `CPluginFactory`, see `BEGIN_FACTORY_DEF` in `pluginfactory_constexpr.h`.

```cpp
class CPluginFactory : public IPluginFactory3,
                       public IPluginFactoryInternal
```

### Global Instance

```cpp
extern CPluginFactory* gPluginFactory;
```

A module-global pointer to the single factory instance. Set by `BEGIN_FACTORY` during the first call to `GetPluginFactory()`. Add-ref is called on subsequent calls via `END_FACTORY`.

## Class Registration

### registerClass (PClassInfo v1)

```cpp
bool registerClass(const PClassInfo* info,
                   FUnknown* (*createFunc)(void*),
                   void* context = nullptr);
```

Registers a class using the v1 class-info struct. `createFunc` is called with `context` when the host requests an instance.

### registerClass (PClassInfo2)

```cpp
bool registerClass(const PClassInfo2* info,
                   FUnknown* (*createFunc)(void*),
                   void* context = nullptr);
```

Registers a class with extended metadata: `classFlags`, `subCategories`, `version`, and `sdkVersion`.

### registerClass (PClassInfoW â€” Unicode)

```cpp
bool registerClass(const PClassInfoW* info,
                   FUnknown* (*createFunc)(void*),
                   void* context = nullptr);
```

Registers a class with UTF-16 strings throughout.

<ParamField name="info" type="const PClassInfo* / PClassInfo2* / PClassInfoW*" required>
  Class descriptor. Usually allocated as a function-local `static`.
</ParamField>

<ParamField name="createFunc" type="FUnknown* (*)(void*)">
  Factory function. Return a new instance of the class (add-refd). The `createInstance` call decrements after querying the requested interface.
</ParamField>

<ParamField name="context" type="void*">
  Arbitrary pointer forwarded to `createFunc`. Defaults to `nullptr`.
</ParamField>

**Returns:** `true` if registration succeeded; `false` if the class table is full and reallocation failed.

## IPluginFactoryInternal

```cpp
class IPluginFactoryInternal : public FUnknown
{
public:
    using HostContextCallbackFunc = void (*)(FUnknown*);
    virtual void PLUGIN_API addHostContextCallback(HostContextCallbackFunc func) = 0;

    static const FUID iid;
};
```

An internal interface that lets plug-in code subscribe to host-context delivery. When the host calls `IPluginFactory3::setHostContext(ctx)`, `CPluginFactory` invokes every registered callback with `ctx`.

### addHostContextCallback

```cpp
void PLUGIN_API addHostContextCallback(HostContextCallbackFunc func) override;
```

<ParamField name="func" type="HostContextCallbackFunc">
  `void (*)(FUnknown*)` called with the host context once `setHostContext` is invoked by the host.
</ParamField>

### Example

```cpp
// In module initialization code:
if (auto* internal = U::cast<IPluginFactoryInternal>(gPluginFactory))
{
    internal->addHostContextCallback([](FUnknown* ctx)
    {
        // ctx is the IHostApplication provided by the host
        MyPlugin::s_hostContext = ctx;
    });
}
```

## Direct Usage (without macros)

While the [Factory Macros](/api/factory-macros) are the recommended approach, `CPluginFactory` can be used directly:

```cpp
SMTG_EXPORT_SYMBOL IPluginFactory* PLUGIN_API GetPluginFactory()
{
    if (!gPluginFactory)
    {
        static PFactoryInfo factoryInfo(
            "My Company",
            "https://my-company.com",
            "mailto:info@my-company.com",
            PFactoryInfo::kNoFlags
        );
        gPluginFactory = new CPluginFactory(factoryInfo);

        static PClassInfo2 processorInfo(
            MyProcessorUID.toTUID(),
            PClassInfo::kManyInstances,
            kVstAudioEffectClass,
            "My Synth",
            Vst::kDistributable,
            "Instrument|Synth",
            nullptr,            // vendor (inherit factory)
            "1.0.0",
            kVstVersionString
        );
        gPluginFactory->registerClass(&processorInfo,
                                       MyProcessor::newInstance);

        static PClassInfo2 controllerInfo(
            MyControllerUID.toTUID(),
            PClassInfo::kManyInstances,
            kVstComponentControllerClass,
            "My Synth Controller",
            0, "", nullptr, "1.0.0", kVstVersionString
        );
        gPluginFactory->registerClass(&controllerInfo,
                                       MyController::newInstance);
    }
    else
    {
        gPluginFactory->addRef();
    }
    return gPluginFactory;
}
```
